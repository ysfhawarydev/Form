# 📝 Form Data Display Project

## 📌 Overview
This is a modern and interactive form project built using HTML, CSS, and JavaScript.  
The form collects user input and redirects to another page where the entered data is displayed dynamically.  
The project focuses on practicing form handling, data transfer between pages, and DOM manipulation.

---

## ✨ Features
- Clean and modern form UI  
- Input validation (basic)  
- Redirect to another page after submission  
- Display submitted data dynamically  
- Responsive design  
- Simple and user-friendly experience  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  

---

## 🎯 Purpose
This project was built to practice JavaScript form handling, data transfer between pages, and improving DOM manipulation skills.
